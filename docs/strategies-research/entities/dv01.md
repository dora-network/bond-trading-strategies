# DV01

**Dollar Value of a Basis Point** — the change in dollar value of a fixed income position for a 1 basis point (0.01%) change in yield.

## Treasury Futures DV01 Reference

| Contract | Instrument | Approximate DV01 |
|----------|-----------|-----------------|
| ZT | 2-Year Note | $38 |
| ZF | 5-Year Note | $47 |
| ZN | 10-Year Note | $65 |
| ZB | 30-Year Bond | $125 |

## Key Applications

### DV01-Weighted Curve Trades
- A 2s10s steepener: buy ZT, sell ZN
- **DV01 ratio:** $65/$38 = 1.71 ZT contracts per 1 ZN for neutrality
- Without DV01 weighting, the position has an unintended directional bias

### Portfolio Hedging
- **Hedge ratio:** Portfolio DV01 ÷ Futures Contract DV01
- Example: $100,000 DV01 portfolio hedged with ZN ($65 DV01) = ~1,538 contracts short

### Duration Target Management
- **Contracts needed:** (Target DV01 − Current DV01) ÷ Futures DV01
- Allows duration adjustment without buying/selling cash bonds

### Risk-Adjusted Position Sizing
- Position size scaled by DV01 to ensure equal risk contribution across strategies
- Critical for multi-strategy portfolio construction

## Duration Conversion
- **Modified Duration:** `DV01 = Price × Modified Duration × 0.0001`
- **Bond DV01:** Sensitive to coupon, maturity, yield level
- **Swap DV01:** Sensitive to curve shape and notional
