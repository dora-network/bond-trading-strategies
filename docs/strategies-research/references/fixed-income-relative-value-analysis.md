# Fixed Income Relative Value Analysis (2nd Ed.)

**URL:** https://www.wiley.com/en-us/Fixed+Income+Relative+Value+Analysis

**Authors:** Doug Huggins, Christian Schaller

## What It Covers

Comprehensive practitioner guide linking financial theory with practical trading tools for fixed income relative value analysis. Updated for SOFR transition.

## Key Claims

1. **Statistical models** essential for quantitative market analysis: mean reversion models and principal component analysis, now including multivariate Ornstein-Uhlenbeck *(extracted)*
2. **Swap spreads** provide key relative value indicators but comparisons are fraught with complications — investors must use caution *(extracted)*
3. **Basis swaps** and their combinations comprehensively discussed as RV tools *(extracted)*
4. **Credit default swaps** can be incorporated into yield curve analysis *(extracted)*
5. **Fitted curve techniques** identify relative value among different bonds — a key tool for rich/cheap analysis *(extracted)*
6. **Multi-factor delivery option model** for bond futures contracts is covered *(extracted)*
7. **PCA** is the primary statistical tool for decomposing yield curve movements into level, slope, and curvature *(inferred)*

## Limitations

- Book format — not a live trading strategy but a reference framework
- Models require adaptation for current market structure post-QE
- Implementation details (coding, data pipelines) left to practitioner
